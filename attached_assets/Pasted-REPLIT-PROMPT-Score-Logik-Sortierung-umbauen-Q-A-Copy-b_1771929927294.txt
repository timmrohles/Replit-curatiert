Ausgangslage

Ziel

2.	User-Signale werden später additiv integriert, aber logarithmisch/normalisiert, damit sie nicht alles überrollen.
3.	„Beliebt“ wird nur angezeigt, wenn ausreichend Signale vorhanden sind (Threshold).
4.	„Geheimtipps“ wird sauber definiert: hohe Basis-Relevanz, aber niedrige Popularität.

A) Implementiere Score-Logik
Phase 1: Base Score (ohne User-Signale)
Berechne pro Buch:
•	award_score:
o	Winner = 10
o	Shortlist = 6
o	Longlist = 4
o	Summe über unabhängige Preise
o	Cap: max 20
•	media_score:
o	pro Medienerwähnung (Podcast/TV/etc.) = 3
o	Cap: max 9
•	curation_score:
o	pro unabhängiger Kuration/Nennung = 2
o	Cap: max 10
•	structure_bonus:
o	Indie (Kurt-Wolff) = +2
o	Debüt = +2
o	Selfpublisher = +1
Base Score:
base_score = award_score + media_score + curation_score + structure_bonus
Phase 2: User Score (wenn genug Daten vorhanden)
•	user_score_raw basiert auf Interaktionen:
o	Like = 1
o	Save = 2
o	Purchase/Outbound-buy = 5 (falls verfügbar)
•	Skaliere/normalisiere mit Logarithmus:
user_score = log(1 + user_score_raw) * 3
(oder eine äquivalente Normalisierung)
Total Score (wenn user_score aktiv):
total_score = base_score + user_score
(oder optional gewichtet: 0.7*base + 0.3*user, wenn wir später wollen)

B) Sortierungen (UI) neu definieren
Implementiere folgende Sorts:
1.	Relevanz (Default)
o	Phase 1: sort by base_score DESC
o	Phase 2: sort by total_score DESC
2.	Neu
o	sort by publication_date DESC (oder „zuletzt hinzugefügt“, wenn das besser passt → dann so benennen)
3.	Am meisten ausgezeichnet
o	sort by award_score DESC
4.	Beliebt
o	sort by user_score DESC
o	nur anzeigen, wenn Mindestschwelle erfüllt (z. B. global >= 200 Likes oder >= 100 Saves oder >= 300 Interaktionen insgesamt; wähle sinnvollen Threshold + Config)
5.	Geheimtipps
o	Filterbedingung: base_score >= X und user_score <= Y (oder wenn Phase 1: curation_score niedrig als Proxy)
o	Sortierung: base_score DESC
o	Ziel: „starke Signale, aber noch nicht Mainstream“
Entferne/ersetze unscharfe Sorts wie „Auszeichnungen“ (das wird zu „Am meisten ausgezeichnet“) und „Hidden Gems“ (wird zu „Geheimtipps“ mit obiger Definition).
